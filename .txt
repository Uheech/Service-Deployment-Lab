0. 브라우저 동작 예시
1) 사용자가 주소창에 주소 입력
2) 브라우저 > DNS 조회> IP로 변환
3) TCP 연결(3-way handshake)
4) https로 암호화된 http 요청 전송(get/)
5) 서버에서 HTML 문서 응답
6) 브라우저가 HTML 해석 후 필요한 CSS/JS/이미지 추가 요청
7) 구글 메인 화면이 렌더링됨 

1. CDN: 콘텐츠 전송 네트워크(Contents Delivery Network)
- 데이터 사용량이 많은 애플리케이션의 웹페이지 로드 속도를 높이는 상호 연결된 서버 네트워크 
- 주 목적: 대기 시간을 줄이고, 네트워크 설계로 인해 발생하는 통신 지연을 줄이는 것 

2. 부트스트랩
<탄생 배경>
- 서로 다른 인터페이스를 사용한 여러 개발자들의 공동작업
- 디자인 불일치, 관리 어려움, 방대한 코드량 > 일관성 유지 불가
- 문제점 개선을 위해 트위터 개발자와 UI 디자이너가 개발
<부트스트랩이란?>
- 프론트엔드 개발을 빠르고 쉽게 할 수 있는 프레임워크
- HTML과 CSS 기반의 템플릿 양식, 버튼, 네비게이션, 기타 페이지를 구성하는 요소 포함
- 자바스크립트를 선택적으로 확장 가능

3. CDN & 부트스트랩
: CDN방식으로 불러오면 설치 없이 바로 사용 가능 

4. rel=stylesheet
- link 태그에 사용되는 속성
- 현재 웹페이지에 연결된 파일이 css파일임을 명시해서 웹 페이지의 디자인과 레이아웃을 제어함
